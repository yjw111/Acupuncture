
1、数据来源：
《当代中国针灸临证精要》、《石学敏实用针灸学》、《中国针灸全书》、《中国当代针灸名家医案》、《针灸治疗学》、《针灸处方学》、《实用针灸临床手册》、《常用临床针灸配穴与禁忌》、《针灸名家医案解读》、《针灸经典处方释义》，以上提及书籍中记录的疾病、证候对应的标准诊疗方案和案例分析记录以及2018年中国国家中医药管理局发布的《中医疾病诊疗方案》中提及的针灸治疗方案。

2、数据处理：
在蒸馏时，所有已知信息都被用于提示，以便约束模型生成合理的思维过程文本。
研究中将方案中仅提及阿是穴或无明确穴位名称记录的数据进行了剔除


注：数据缺少临床真实采集样例，真实样例为合作医院单位提供，本研究无权公开。
